# Version Checker

## Dev setup
```bash
sudo apt install python3 python3-venv python3-pip
python3 -m venv .venv
. .venv/bin/activate
pip install -e .[dev]
python3 -m build
```

## Install
```bash
sudo apt install python3 python3-pip
python3 -m pip install version_checker-0.0.1-py3-none-any.whl
```
